Gary Hillerson, M.A.

Technical Writing and Information Architect Experience

Senior Technical Writer, Splice Machine, 2014-present

I was hired as a contractor in 2013 at Splice Machine to develop documentation for their nascent big data relational database system. I became a full-time employee at Splice Machine in March, 2014. Since then, I have been the sole developer and writer for the company, creating a suite of five books comprising almost one thousand topics:

  • Getting Started with Splice Machine
  • Developer's Guide to Splice Machine
  • Splice Machine Administrator's Guide
  • Installation Guide for Splice Machine
  • Splice Machine SQL Reference Manual

We've used MadCap Flare as our writing tool, which has allowed us to easily package all of the books into a single, responsive web site, doc.splicemachine.com, and to also offer each book as a downloadable PDF.

Writing the documentation for Splice Machine has primarily involved working with the developers, and meeting with product management and customer solutions managers to help shape the design and delivery of the content.
