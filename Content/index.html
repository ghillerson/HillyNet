Gary Hillerson, Technical Writing Expert

Expert Documentation Architecture, Design, and Writing Services

I am an information architect, a technical writer, and an ex-software engineer. I've lived in the Santa Cruz, California area since 1975. I enjoy organizing chaotic information into structures that make sense to whatever audience is using that information. I think of technical writing as a different version of programming: one in which the programming language is English, and the classes and modules are chapters and topics.

In recent years, I've become an expert in using MadCap Flare for various writing and web development projects, including an extensive documentation system for a Big Data database company, an online learning course in Music Theory, and the web and help system for an academic software company. Prior to using Flare, I gained proficiency in DreamWeaver, FrameMaker, and Word.

Technical Writing and Information Architect Work History

In 1989, I switched from programming to writing technical documentation for software engineers. The reason was quite simple: I was burning out from my compulsive relationship to solving coding problems, and realized that I needed to find a way to relativize my relationship to software. I've been glad of it every since.
